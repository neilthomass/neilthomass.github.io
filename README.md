# neilthomass.github.io

Personal site powered by [Next.js](https://nextjs.org/) and TypeScript.

## Requirements

- Node.js 18 or later

## Installation

```bash
npm install
```

## Development

Start the development server:

```bash
npm run dev
```

## Build

Create a production build:

```bash
npm run build
```

Run the built application locally:

```bash
npm start
```

## Linting

```bash
npm run lint
```
